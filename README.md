# InsuredMine Backend Assessment

This project is developed as part of the InsuredMine backend developer assessment.

## Features
- Upload insurance data via CSV/XLSX
- Background worker thread for data processing
- Search policies by username
- Aggregate policies per user
- Monitor CPU usage and restart server on high load
- Schedule messages for a specific date and time

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Worker Threads
